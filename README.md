# Next.js Project

## **Getting Started**

### **Installation**

1. **Clone the repository:**
   ```bash
   git clone https://github.com/oGuilherme1/test-logzz-front.git
   ```
   
   ```bash
   cd test-logzz-front
   ```
   
   ```bash
   npm install
   ```
   
   ```bash
   npm run dev
   ```

  ### **Open your browser and go to http://localhost:3000.** ###
